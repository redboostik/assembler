# ASM Starter Template
ASM Starter templete
## Structure
- data
  - INPUT.TXT
  - OUTPUT.TXT
  - STARTUP.TXT
- debug
  - COMPILE.TXT
  - LINK.TXT
- main.asm
- Makefile
- sio.asm
## Usage
1. Download TASM and TLINK
2. Configure Makefile
3. Install DOSBox
```bash
sudo apt-get install dosbox
```
4. Execute:
```bash
make all
```
